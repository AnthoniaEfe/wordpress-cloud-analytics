A cloud-hosted web app that connects to one or more WordPress sites using their REST API, collects data (like traffic, engagement, performance, and SEO stats), and presents it in a clean, real-time dashboard.

Frontend: React + Tailwind (data visualization, UI)

Backend: Node.js + Express (API + logic + integration)

Database: MongoDB (storing users, site configs, analytics data)

Cloud Deployment: Vercel (frontend) + Render or AWS (backend) + MongoDB Atlas (database)
